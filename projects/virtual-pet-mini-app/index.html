Virtual Dog - Demo

MiniVirtualDog is a Java-based demo application that simulates a simple virtual dog. It serves as a prototype implementation with basic interactive features, though some functionality and graphics are still incomplete or simplified. The project includes core classes for Dog, user Controls, and display Panel logic, demonstrating object-oriented design and real-time animation handling in Java. The primary goal of this repository is to showcase a lightweight graphical demo rather than a fully polished game, illustrating foundational programming techniques in Java graphics and control flow for future expansion.
