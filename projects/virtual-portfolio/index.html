Virtual Portfolio

Virtual Portfolio is a self-designed, fully deployed web application created to serve as a centralized hub for my professional experience, technical research, and engineering projects. Built using HTML5, Tailwind CSS, vanilla JavaScript, GitHub, and Cloudflare Pages, the platform emphasizes clean architecture, modular structure, and long-term scalability rather than functioning as a simple static resume site. It integrates project documentation, research work, blog content, and technical case studies into a cohesive, evolving system that reflects both my software development capabilities and systems-engineering mindset. Designed to grow alongside my career, Virtual Portfolio serves as both a technical demonstration and a continuously improving professional platform.
